# Laboratorio de Ejercicios en Java

Este repositorio contiene un programa en Java que integra múltiples ejercicios de programación en un solo proyecto, aplicando conceptos fundamentales como modularización, estructuras de control, validación de datos, manejo de excepciones, simulación de paso por referencia y buenas prácticas de organización.

## Características

- **Menú interactivo** para elegir entre 10 ejercicios diferentes.
- **Validación de entradas numéricas** con `try-catch` (similar a `TryParse` en C#) para evitar caídas del programa.
- **Uso de ciclos**: `while`, `do-while` y `for` según la necesidad.
- **Modularización**: cada ejercicio está implementado en su propio método, y las operaciones comunes en funciones auxiliares.
- **Simulación de paso por referencia** en Java mediante arreglos.
- **Código indentado y organizado**, con nombres descriptivos de métodos y variables.

## Ejercicios incluidos

1. **Calculadora básica**  
   - Suma, resta, multiplicación y división.  
   - Validación de división entre cero.

2. **Validación de contraseña**  
   - Solicita una contraseña hasta que sea `"1234"`.  
   - Uso de ciclo `do-while`.

3. **Números primos**  
   - Función que determina si un número es primo.  
   - Muestra el resultado en el programa principal.

4. **Suma de números pares**  
   - Acumula números pares ingresados hasta que se ingrese `0`.  
   - Uso de ciclo `while`.

5. **Conversión de temperatura**  
   - Conversiones: Celsius ↔ Fahrenheit.  
   - Métodos separados para cada conversión.

6. **Contador de vocales**  
   - Recibe una cadena y cuenta cuántas vocales contiene (incluyendo tildes).

7. **Cálculo de factorial**  
   - Calcula el factorial de un número entero no negativo.  
   - Validación de entrada negativa.

8. **Juego de adivinanza**  
   - Genera un número aleatorio entre 1 y 100.  
   - Pista de "mayor" o "menor" hasta acertar.  
   - Ciclo `do-while`.

9. **Intercambio de números (simulación paso por referencia)**  
   - Intercambia dos valores usando un arreglo para simular el paso por referencia.

10. **Tabla de multiplicar**  
    - Muestra la tabla de multiplicar del 1 al 10 para un número dado.

## Requisitos

- Java Development Kit (JDK) 8 o superior.
- Terminal o entorno de desarrollo (IDE) para compilar y ejecutar.
